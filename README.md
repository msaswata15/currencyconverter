
# Currency Converter

A React app to convert between different currencies using live exchange rates. Styled with Tailwind CSS.

## Features
- Real-time currency conversion
- Swap currencies with a floating button
- Responsive, modern UI

## Getting Started
1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm start
   ```

## Tech Stack
- React
- Tailwind CSS
- [Fawaz Ahmed Currency API](https://github.com/fawazahmed0/currency-api)
